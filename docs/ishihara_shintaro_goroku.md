# 石原慎太郎 語録

以下は、提供された談話を文章として整えた記録です。

人間というのは面白いもので、結婚という制度に縛られているようでいて、その中でいちばん自由を求めているのもまた人間だ。私はこれまで政治家、作家、俳優、庶民、金持ち、貧乏人と、さまざまな人々を見てきたが、ある時ふと思うようになった。この人は一人でいた方がずっと幸せなのではないかと。もちろん結婚が悪いわけではない。家庭を持つことで人間が磨かれることもある。誰かと生きるというのは己の我を削り、愛という不思議な力で形を変える作業でもある。だが全員がそれをやる必要はない。最初から自由であることこそ自分の呼吸のような人間がいて、そういう人間は他人と一緒に暮らすと息苦しくなる。愛していないわけではない。ただ、他人の時間に合わせることがどうしてもできないのだ。

私の周りにも、仕事では抜群の才能を発揮するのに恋愛となると途端に不器用になる男がいた。理由を尋ねると、自分の時間を削ってまで誰かに気を使うのが嫌なのだと言う。冷たく聞こえるかもしれないが、その気持ちは痛いほどわかる。人間は誰かと共にいるためには自分の一部を差し出さなければならない。元々、自分という存在を生きることに全てをかけている人間にとって、それは魂の切除手術のようなものだ。結婚に向いていない人間は決して愛を知らないわけではない。むしろ愛というものの重みを知りすぎている。だからこそ、自分が誰かの人生に入り込み、その人の時間を狂わせることを恐れてしまう。

独身の方が幸せな人というのは、孤独を孤独と思っていない。夜に一人で食事をしても寂しいと感じない。むしろ、誰にも邪魔されずに考え、静かに自分の内側を見つめる時間を愛おしむ。私も若い頃に結婚して家庭を持ったが、正直な話をすれば、家庭と創作、この二つを同時に満たすことはほとんど不可能に近いと思っている。作家というのは結局孤独を糧にする職業で、誰かと笑い合っている時間にもどこか頭の片隅では次の作品のことを考えてしまう。それはもう性分なのだ。そんな自分を理解してくれる相手がいればよいが、ほとんどの場合は理解されない。「あなたは私を大切にしてくれない」「いつもどこか別の世界を見ている」と言われるたびに罪悪感に苛まれる。しかし罪悪感というのは必ずしも悪いものではない。人を傷つけたくないと思うその気持ちは、人間の良心だからだ。

良心を持ちながらも、自分は誰かと生きるより一人でいた方が正直でいられると悟る人間もいる。私はそういう人間を否定するつもりは全くない。むしろ、そういう人間こそが社会の中で真に自由というものを理解しているのだと思う。自由とは誰にも依存しないということでも、他人を支配しようともしないことでもない。その境界線を自分の中で引けるかどうかが大切なのだ。結婚をすれば当然そこには約束が生まれる。約束は大切だが、人によってはその約束が首を絞める。愛しているのに息が詰まる。そんな矛盾を抱えて生きる人間を私は何人も見てきた。

人間というのは弱い生き物で、誰かに認められたい、愛されたい、必要とされたいと思う。それは当然だ。だが、一人で幸せを感じられる人間は、その欲求をどこかで乗り越えてしまっている。誰かに理解されなくてもいい。自分で自分を理解できればそれでいい。そう思える人間だ。そしてそういう人間こそ本当の意味で結婚に向いていないのではなく、結婚という形に収まらないだけなのだ。一人でいることを選んだ人の中には、自由でいたいというよりも自分の世界を壊されたくないという思いの方が強い者がいる。人間というのはどんなに社交的に見えても、心の奥には誰にも触れられたくない領域がある。その領域が広い人ほど、一人で生きることを苦にしない。

若い頃の私は、人と関わることが面倒でたまらなかった。宴会も嫌いだった。群れることに何の価値も感じなかった。自分の考えが誰かの考えに染まっていくのが何より怖かった。だからこそ私は一人の時間を好んだ。自分の考えを言葉にするためには、他人の声を全て遮断する必要がある。書くという行為は孤独と対話することだ。だが皮肉なことに、そうやって孤独を生きる人間ほど、他人からは冷たい、近寄りがたいと言われる。本当は違う。人よりもずっと深く繊細に人を見ている。ただその優しさを安易には与えないだけなのだ。

世の中には「誰かのために生きることこそ人間の本懐だ」という言葉がある。だが私はそんな言葉に簡単に頷くことはできない。誰かのために生きることが必ずしも善ではない。自分を犠牲にしてまで人を支えることは、時に自分という存在への裏切りになる。結婚というのはまさにそういう善意の契約だ。互いに支え合うという名のもとに、少しずつ自分を削っていく。だがその削り方を間違えると、いつの間にか何のために生きているのかわからなくなる。

私が見てきた中で独身のまま幸せに生きている人間には共通点がある。それは自分の時間を守るために戦っているということだ。時間というのは命そのものだ。誰かと暮らすというのはその命を分け合うということだ。だが、自分の命を作品を作ること、表現すること、考えることに使いたい人間にとって、その時間の共有は時に拷問にも等しい。たとえば夜中に突然アイデアがひらめき、それを書き留めようと机に向かう。そんな時に「もう寝なさいよ」と声をかけられると、たとえ優しさであっても心のどこかで怒りが芽生える。想像の炎を他人の都合で消されたように感じてしまうのだ。

もちろんそれはわがままだ。だが人間というのはわがままであるからこそ面白い。全員が良い人間を演じていたら、この世は退屈で仕方がない。私は昔から、自分の感情に正直であることこそが誠実だと思っている。愛していると言いながら心の中で窮屈さを感じているなら、それはもう嘘だ。そして自分に嘘をついた瞬間、人間は輝きを失う。結婚に向かない人というのは、そうした内なる誠実さが人一倍強い。他人を裏切るくらいなら最初から約束を交わさない。そんな人間だ。社会はそういう人を冷たい、自己中心的と批判する。だが私はそういう人間を尊敬している。なぜなら彼らは孤独と共に立つ覚悟を持っているからだ。

孤独は怖い。誰かに必要とされない時間というのは時に自分の存在を疑うほどの痛みを伴う。それでも自分の生を生き抜く覚悟がある人間は、最後に他人ではなく自分に認められる。私が人生で最も苦しかったのは孤独そのものではなく、他人の期待に応えようとして自分を失った時だった。誰かのために頑張るのは立派だ。だがそれが義務になった瞬間、心は乾く。愛が義務に変わった時、人は息ができなくなる。

結婚というのは本来、愛の延長にあるべきものだ。だが現実は社会的責任や世間体に縛られることが多い。その構造に息苦しさを感じる人間は、最初から独身でいる方が正直だ。結婚をしないという選択は、愛を信じないということではなく、愛を純粋なまま汚したくないという祈りのようなものだ。人は皆愛を求める。だがその愛の形は一つではない。誰かと共に生きる愛もあれば、一人で生きること自体が愛になる場合もある。自分を大切にするという愛。それを貫ける人間は強い。

人間というのは孤独を恐れるあまり、誰かと繋がっていないと不安になる。だが本当に強い人間というのは孤独を共にできる人間だ。孤独を恐れない人間はまず自分の内側と向き合う覚悟を持っている。人間は誰しも自分の中に醜さを抱えている。嫉妬、虚栄、怠惰、弱さ。そういうものを直視できず、他人といることでごまかして生きている人間は多い。だが一人の時間は残酷だ。自分の弱さが鏡のように映る。その時間に耐えられない人間は、すぐに他人の声で自分を紛らわせようとする。しかしそれを乗り越えた先にしか、本当の意味での自由はない。

自由とは好き勝手に生きることではない。他人の承認がなくても自分の価値を信じて生きられることだ。私は若い頃からずっとそう思ってきた。周囲にどう思われようが、自分の考えを曲げるのがいちばんの罪だと感じていた。だからこそよく反感を買った。だが、それでいい。誰にも嫌われないように生きるというのは自分を薄めていく行為だからだ。結婚に向かない人間の多くは、この自分を薄めることへの拒絶が強い。自分の世界を守るということは、同時に他人の世界を壊さないということでもある。だから一人を選ぶ。それは逃避ではなく選択なのだ。

ある女優がこう言っていた。「私は結婚しないけれど、愛していないわけじゃない。ただ、誰かの隣にいながら自由でいる自信がないだけ」。私はその言葉を聞いて深く頷いた。人間には、他人と一緒にいても自由でいられる人間と、一人でいるからこそ自由でいられる人間がいる。どちらが正しいということではない。だが後者の人間が結婚すれば、いずれどちらかが不幸になる。だからこそ最初から一人を選ぶというのは、ある意味で他人を不幸にしないための優しさでもあるのだ。

世の中は結婚、幸せ、家庭、安定と刷り込んでくる。だがそんなものは幻想だ。人によって幸せの形は違う。他人と生きることが幸せな人もいれば、一人で生きることが幸福な人もいる。そして後者は決して少数派ではない。私は、誰かといなければ孤独だと思っている人ほど本当の孤独を知らないと思う。孤独というのは悪いものではない。人間を鍛える時間だ。誰かの期待も視線も評価もない場所で、自分の思考と感情を研ぎ澄ます。それはある意味で魂の筋トレだ。孤独を味方につけた人間は他人に優しくなれる。なぜなら自分の弱さを知っているからだ。

自分がどれほど脆く未完成な存在かを知っているから、他人を許せる。逆に常に誰かと群れていないと安心できない人間ほど他人に厳しくなる。それは自分を守るために他人を裁いているからだ。孤独を恐れない人間は静かだが、その静けさの中には芯がある。風が吹いても折れない、根を深く張った木のような強さがある。私はよく思う。本当の魅力というのは孤独を知っている人間からしか生まれない。群れの中で目立つ人間より、一人で立っている人間の方がずっと人を引きつける。なぜならその背中には覚悟があるからだ。

結婚しても家庭を持っても覚悟のない人間は多い。責任感と覚悟は似ているようで違う。責任感は他人のために動く力だが、覚悟は自分のために立ち続ける力だ。独身で幸せな人というのは、この覚悟を自分の中に持っている。だから誰かに依存しない。誰かに期待もしない。それでいて他人の幸せを素直に喜べる。それが本当の大人だ。

私は若い人たちに伝えたい。結婚は人生の義務ではない。無理に形を作る必要はない。一人で飯を食い、一人で笑い、一人で考える。それを寂しいと思ううちは、まだ自分を知らない。だが、それを豊かだと思えるようになった時、ようやく自分の人生を自分の手に取り戻したということだ。人は最後の瞬間を迎える時、誰といるかよりもどんな自分でいられるかを思うものだ。どんなに愛されてきたとしても、自分を偽り続けた人生なら心は安らがない。独身でいるというのは、その偽りを最小にする生き方だと思う。誰の期待にも理想にも合わせない。自分の足で立ち、自分の責任で選び、失敗しても自分で受け止める。そういう人間は最後の瞬間に他人を責めない。

世の中には「一人は寂しい」と口にする人がいる。だが本当に寂しいのは、誰かと一緒にいながら孤独を感じることだ。隣に人がいても心が通じ合わない。その方がよほど辛い。一人で生きることを恐れない人間はその痛みを知っている。だからこそ本当の繋がりを求める。そしてそこに妥協しない。私は若い頃から人間関係においても、無理をしてまで人と繋がる必要はないと思っていた。自分を消してまで続ける関係はいずれどこかで破綻する。他人の顔色をうかがっているうちは、本当の意味で愛されることはない。

独身を貫く人の中には、愛されなかった人ではなく、愛を安売りしなかった人がいる。誰にでも心を開くのではなく、本当に価値を感じた相手にしか心を渡さない。だからこそ彼らの愛は深い。一瞬であっても本気で愛した記憶がある人間は、一生一人で生きても空虚ではない。結婚して幸せになる人もいる。だが一人で幸せになる人はもっと少ない。なぜなら自分の中に家を建てるという作業は、他人と生きるよりもはるかに難しいからだ。

人は誰でも心のどこかで誰かに理解されたいと願っている。だがそれを完全に叶えることはできない。たとえ何十年連れ添った夫婦でも、心の奥底まで分かり合うことはない。結局のところ人間は皆孤独な存在だ。だがそれでいい。孤独であることを前提にした時、人はようやく他人に優しくなれる。相手に全てを求めない。分かり合えないことを受け入れる。それが成熟というものだ。

結婚というのは、ある意味で永遠の錯覚を共有する契約だ。だがその錯覚を現実と調和させられるのはごく一部の人間だけだ。大半は理想と現実の狭間で疲弊する。私は孤独の中にも幸福はあると言いたい。いや、むしろ孤独の中でしか得られない幸福がある。朝、静かな部屋でコーヒーを入れる音。夜、誰にも邪魔されずに本を読む時間。自分の思考が自分だけのものであるという誇り。それがどれほど贅沢なことか。

社会は人を型にはめたがる。結婚しなさい、家庭を築きなさいと。だが人生というのは肩書ではなく風のようなものだ。吹く方向を自分で選べる人間が、最後にはいちばん穏やかに生きられる。一人でいることを選んだ人間は寂しさの代わりに誇りを手に入れる。他人に評価されなくても、自分の人生を自分で肯定できる。それができた時、人はようやく自由になる。私はそういう生き方をした人間を尊敬している。どんなに世間から奇異に見られようと、自分の信じた道を歩く者は美しい。

最後にこう言いたい。独身でいることを恐れるな。それは自分を大切にできるという証拠だ。誰かと生きることも素晴らしい。だが誰とも生きずに、それでも人生を豊かに燃やせる人間はもっと素晴らしい。孤独を恐れず、自分の心を最後まで裏切らないでほしい。人は誰かに愛されて幸せになるのではない。自分の生き方に納得できた時、その瞬間に幸せになるのだ。
